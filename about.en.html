    Resume files are encoded in base64. Sample Python code to decode them:

    
import base64

with open(downloaded_filepath, 'rb') as f:
    content = f.read()

unwrapped_content = b''.join(content.split(b'\r\n'))
decoded_content = base64.b64decode(unwrapped_content)

with open(output_filepath, 'wb') as f:
    f.write(decoded_content)

    Please take a look at encoding.py to find out how they are encoded.
    Note: At least Python 3.6 is required to run this script.
