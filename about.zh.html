简历文件经过了 base64 编码，请下载后自行解码。

比如使用 Python 代码解码简历文件：


import base64

with open(downloaded_filepath, 'rb') as f:
    content = f.read()

unwrapped_content = b''.join(content.split(b'\r\n'))
decoded_content = base64.b64decode(unwrapped_content)

with open(output_filepath, 'wb') as f:
    f.write(decoded_content)

如需了解这些简历文件是如何编码生成的，请查阅 encoding.py 。
注：运行这个脚本需要的 Python 版本至少是 3.6 。
